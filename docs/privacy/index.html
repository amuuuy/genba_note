プライバシーポリシー

施行日: 2026年3月1日（最終更新: 2026年4月23日）

1. はじめに

本プライバシーポリシーは、ポチッと事務（以下「本アプリ」）における利用者の情報の取り扱いについて説明します。本アプリは「ローカルファースト」の設計思想に基づき、利用者のデータを可能な限り端末内に保存し、外部サーバーへの送信を最小限に留めています。v1.0.1 以降、すべての機能を無料でご利用いただけます（アプリ内課金は廃止しました）。

2. 収集する情報

2.1 端末内に保存される情報（外部送信なし）

以下の情報は利用者の端末内にのみ保存され、開発者のサーバーには送信されません。保存場所は情報の種類により以下の3つに分かれます。

(a) 通常ストレージ（AsyncStorage）に保存される情報（テキストデータ・URI 参照のみ）

  • 見積書・請求書データ（書類番号、品目、金額等）
  • 顧客情報（名前、住所、連絡先等）
  • 単価マスタデータ
  • 収支記録
  • 予定データ
  • 発行者情報のテキスト項目（会社名、代表者名、住所、電話番号、FAX、メールアドレス、書類番号プレフィックス等）
  • アプリ設定（テンプレート選択、印影サイズ、背景デザイン等）
  • 写真メタデータ（顧客写真・領収書写真の URI 参照、ファイル名、撮影日時等）

(b) セキュアストレージ（SecureStore: iOS Keychain / Android Keystore）に暗号化保存される情報

  • 適格請求書発行事業者番号（インボイス番号）— 現在の発行者設定値
  • 銀行口座情報（銀行名・支店名・口座種別・口座番号・口座名義）— 現在の発行者設定値
  • 書類ごとの機密スナップショット — 見積書・請求書を作成した時点の適格請求書発行事業者番号と銀行口座情報を、書類IDをキーとして個別に保存します。書類を削除すると、対応するスナップショットも削除されます

(c) アプリ専用ディレクトリに保存される情報（画像ファイル本体）

  • 印影画像（社印などのスタンプ画像）
  • 書類の背景画像
  • 顧客ごとの工事写真
  • 収支記録の領収書写真

(c) の画像ファイル本体はアプリ専用ディレクトリ（iOS の Documents Directory / Android のアプリ内ストレージ）に保存され、(a) の AsyncStorage には対応する URI のみが記録されます。

2.2 外部サービスに送信される情報

本アプリで予期しないエラー（JavaScript クラッシュまたは ErrorBoundary により捕捉された React レンダリングエラー）が発生した場合に限り、以下の匿名化された情報が外部サービスに送信されます。通常の業務データ（書類・顧客・写真等）は送信されません。

外部サービス 送信される情報 目的
Sentry 匿名化されたエラーレポート（端末情報、エラー内容、スタックトレース、コンポーネントスタック） アプリの品質改善

パフォーマンストレース、検索クエリ、購入情報、認証情報など、エラーレポート以外のデータは一切送信しません（tracesSampleRate は 0 に設定されており、通常動作時の Sentry 送信はありません）。

2.3 v1.0.0 からの変更点

v1.0.0 では、以下の外部サービスとの連携がありました。v1.0.1 ですべて廃止しています。

  • Supabase（匿名認証・Edge Functions）— 廃止
  • RevenueCat（サブスクリプション管理）— 廃止（Proプラン自体を廃止）
  • Google Gemini API（AI資材検索）— 廃止
  • 楽天市場 API（資材検索）— 廃止

v1.0.0 の利用中にこれらのサービスに送信された情報の削除をご希望の場合は、下記お問い合わせ先までご連絡ください。

3. アカウント登録について

本アプリはアカウント登録なしでご利用いただけます。メールアドレス、パスワード等の認証情報の入力は不要です。

4. 情報の保存場所

情報の種類ごとの保存場所の詳細は「2.1 端末内に保存される情報」をご参照ください。概要は以下の通りです。

  • AsyncStorage（通常の端末内ストレージ）: 業務データ（書類・顧客・単価・収支・予定）、発行者情報のテキスト項目、アプリ設定、画像の URI / メタデータ
  • SecureStore（iOS Keychain / Android Keystore）: 機密性の高い情報として、適格請求書発行事業者番号と銀行口座情報を暗号化して保存します。具体的には (1) 現在の発行者設定値と (2) 各書類作成時点のスナップショット（書類削除と連動して削除）の2種類を保持
  • アプリ専用ディレクトリ: 印影画像・背景画像・顧客写真・領収書写真などの画像ファイル本体

5. 個人情報の保護

開発者は、個人情報の保護に関する法律（個人情報保護法）を遵守し、利用者の個人情報を適切に管理します。開発者が利用者の個人情報を直接取得することはありません（上記の外部サービスへの送信を除く）。

6. データの保持期間

  • 端末内データ: アプリのアンインストールまで保持されます。
  • Sentry エラーレポート: Sentry のデフォルト保持期間（90日）に準じて自動削除されます。

7. データの削除

以下の方法でデータを削除できます。

  • 個別削除: アプリ内の各画面から書類・顧客・写真等を個別に削除
  • 全データ削除: 本アプリをアンインストールすることで、端末内のすべてのデータが削除されます
  • Sentry エラーレポートの削除: 特定のエラーレポートの削除を希望される場合は、下記お問い合わせ先までご連絡ください。対応可能な範囲で削除手続きを行います

8. 児童のプライバシー

本アプリは13歳未満の児童を対象としていません。13歳未満の方が本アプリを利用していることが判明した場合は、お問い合わせください。

10. 改定

開発者は本プライバシーポリシーを随時改定する場合があります。改定後のポリシーは本ページに掲載した時点で効力を生じます。重要な変更がある場合は、アプリ内での通知を行います。
